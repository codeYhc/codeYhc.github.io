安装 patch-package ​

首先，我们需要安装 patch-package 和 postinstall-postinstall，它们将帮助我们生成和应用补丁。

npm i patch-package postinstall-postinstall --save-dev

配置 package.json ​

在 package.json 中添加 postinstall 脚本，以便在每次依赖安装后自动应用补丁。

{
  "scripts": {
    "postinstall": "patch-package"
  }
}

创建补丁 ​

假设我们需要修改一个依赖包 some-package。我们可以按照以下步骤创建补丁：

  1. 修改依赖包

在 node_modules/some-package 中找到需要修改的文件，并进行必要的修改。例如，我们要修改 index.js 文件。

  1. 生成补丁

运行以下命令生成补丁文件：

patch-package some-package

这将在项目根目录下生成一个 patches 文件夹，其中包含生成的补丁文件，例如 patches/some-package+1.0.0.patch。

确保补丁文件被添加到版本控制系统 ​

确保补丁文件被添加到你的版本控制系统（例如 Git）。

git add patches
git commit -m "Add patch for some-package"

自动应用补丁 ​

每次运行 pnpm install 时，postinstall 脚本会自动运行 patch-package，从而应用补丁。
